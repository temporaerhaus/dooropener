# dooropener

Firmware for the Olimex ESP32-EVB/ESP32-POE-ISO and PN532 based dooropener.

## Setup

 1. Adjust `include/config.h` to your needs
 1. Flash boards with platform.io (`pio run -e esp32-poe-iso|esp32-evb`)
 1. Profit
